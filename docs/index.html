DOCX to Markdown

Upload a .docx and download a Markdown + media zip from GitHub Actions.

1. Connect to GitHub

Token permissions: Actions (read/write). Fine-grained PAT recommended.

2. Upload DOCX

Uploads to a temporary file host and triggers conversion. No files are committed to the repo.

3. Download output

Downloads the most recent artifact zip and loads it into the editor.

4. Edit & Preview

Preview

Edits update the preview instantly. Image links are resolved from the downloaded media files.
